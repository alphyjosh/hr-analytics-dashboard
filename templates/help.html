How to Use the HR Analytics Dashboard

Getting Started

Welcome to the HR Analytics Dashboard! This web application allows you to track and visualize monthly staffing data, including headcount, new joiners, leavers, and attrition rates.

Key Features

  • Interactive Dashboard: View key metrics, charts, and tables of your staffing data
  • Custom Inputs: Add your own staffing values through the Edit Data page
  • Customizable Branding: Change colors and company information
  • Data Export: Download your data in CSV format
  • Forecasting: View projected headcount based on historical trends

How to Edit Your Staffing Data

  1. Navigate to the Edit Data page
  2. Modify the existing values or add new months using the "Add Month" button
  3. Click "Save Changes" to update the dashboard
The application comes with sample data to help you get started. Use the "Reset to Defaults" button to restore this sample data at any time.

Understanding the Charts

  • Monthly Headcount Trend: Shows how your total employee count changes over time
  • Monthly Staff Movement: Compares new joiners vs leavers each month
  • Monthly Attrition Rate: Shows the percentage of employees leaving each month (red line indicates the 5% warning threshold)
  • Staffing Forecast: Projects future headcount based on current trends (requires at least 3 months of data)

Customizing Your Dashboard

Visit the Branding page to customize:

  • Company name
  • Dashboard title
  • Color scheme

Sharing Your Dashboard

To make this dashboard accessible to others:

  1. Deploy this application to a web hosting service like Heroku, Render, PythonAnywhere, or Netlify
  2. Share the deployed URL with your team members
  3. Anyone with the link can view and interact with the dashboard
Remember that the data is stored in session memory. When deployed to most platforms, data will reset if the server restarts. For persistent storage, consider adding a database.
